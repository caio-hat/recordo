## Acknowledgements

Recordo builds on top of excellent open-source projects:

- [GTK4](https://www.gtk.org/) and [libadwaita](https://gnome.pages.gitlab.gnome.org/libadwaita/) — UI toolkit
- [faster-whisper](https://github.com/SYSTRAN/faster-whisper) — Whisper inference (CTranslate2)
- [sherpa-onnx](https://github.com/k2-fsa/sherpa-onnx) — Parakeet ONNX inference
- [NVIDIA NeMo](https://github.com/NVIDIA/NeMo) — original Parakeet model
- [Ollama](https://ollama.com/) — local LLM serving
- [Textual](https://textual.textualize.io/) — terminal UI
- [markdown-it-py](https://github.com/executablebooks/markdown-it-py) — Markdown parser
- [Pygments](https://pygments.org/) — syntax highlighting
- [WebKitGTK](https://webkitgtk.org/) — embedded HTML rendering

Special thanks to:

- [istupakov](https://huggingface.co/istupakov) for the ONNX export of Parakeet TDT 0.6B v3
- [jlondonobo](https://huggingface.co/jlondonobo) for the pt-BR Whisper fine-tune
- [achetronic/parakeet](https://github.com/achetronic/parakeet) and [cjpais/Handy](https://github.com/cjpais/Handy) for prior art on lightweight Parakeet inference
